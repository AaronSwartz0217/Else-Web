Unity 3D 开发

学习了 Unity 引擎的基本使用，包括场景构建、脚本编写、物理系统等。通过实践项目掌握了游戏开发的核心流程，从概念设计到最终发布。

Shader 编程

重点学习了 Shader 编程，能够创建各种视觉效果，如水面、火焰、卡通渲染等。

// 简单的水面Shader示例 Shader "Custom/WaterShader" { Properties { _MainTex ("Texture", 2D) = "white" {} _WaveSpeed ("Wave Speed", Range(0.1, 10)) = 1.0 _WaveHeight ("Wave Height", Range(0.01, 0.5)) = 0.1 } SubShader { Tags { "RenderType"="Opaque" } LOD 100 Pass { CGPROGRAM #pragma vertex vert #pragma fragment frag #include "UnityCG.cginc" struct appdata { float4 vertex : POSITION; float2 uv : TEXCOORD0; }; struct v2f { float2 uv : TEXCOORD0; float4 vertex : SV_POSITION; }; sampler2D _MainTex; float4 _MainTex_ST; float _WaveSpeed; float _WaveHeight; v2f vert (appdata v) { v2f o; // 添加波浪效果 v.vertex.y += sin(_Time.y * _WaveSpeed + v.vertex.x) * _WaveHeight; o.vertex = UnityObjectToClipPos(v.vertex); o.uv = TRANSFORM_TEX(v.uv, _MainTex); return o; } fixed4 frag (v2f i) : SV_Target { fixed4 col = tex2D(_MainTex, i.uv); return col; } ENDCG } } }

Unreal Engine 5

探索了 UE5 的蓝图系统和材质编辑器，学习了如何创建高质量的视觉效果。掌握了蓝图逻辑编程，能够实现复杂的游戏机制。

UE5 新特性

研究了 UE5 的新特性，如 Lumen 全局光照和 Nanite 虚拟几何体，了解了如何利用这些技术提升游戏的视觉质量。

// 实现物体自转 // 创建类，使用Actor类新建蓝图把旋转物体作为根组件，然后加入旋转的类组件 // 头文件 // SelfRotator.h // 防止头文件被多次包含 #pragma once // 包含虚幻核心类型（如FString, Array, FVector等） #include "CoreMinimal.h" // 包含 ActorComponent 基类的头文件，因为我们要继承它 #include "Components/ActorComponent.h" // 由 UnrealHeaderTool 自动生成的反射代码，必须放在最后 #include "SelfRotator.generated.h" /** * 这是一个可以加到任何 Actor 上的组件，使 Actor 自动绕自身 Z 轴旋转。 * 旋转速度可以在编辑器中调整，也可以被蓝图读写。 * 该类可以被 C++ 或蓝图继承，以扩展功能。 */ UCLASS(ClassGroup = (Custom), meta = (BlueprintSpawnableComponent)) class UE_CPP_API USelfRotator : public UActorComponent { GENERATED_BODY() public: // 构造函数 USelfRotator(); protected: // 组件启动时调用一次（在游戏开始或 Actor 生成后） virtual void BeginPlay() override; public: // 组件每帧更新时调用 virtual void TickComponent(float DeltaTime, ELevelTick TickType, FActorComponentTickFunction* ThisTickFunction) override; // 旋转速度（度/秒） UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Rotation") float RotationSpeed = 45.0f; };

游戏设计原理

学习了游戏设计的基本原理，包括游戏机制、关卡设计、用户体验等。通过分析经典游戏，理解了优秀游戏的设计思路。

游戏心理学

研究了游戏心理学，了解玩家的行为模式和心理需求，能够设计更具吸引力的游戏体验。

网页交互设计

学习了现代网页交互设计的 principles，包括响应式设计、用户界面设计、交互模式等。掌握了 HTML、CSS 和 JavaScript 的高级应用。

Three.js

研究了 Three.js 库，能够创建交互式 3D 网页，为用户提供沉浸式的浏览体验。

用户体验设计

学习了用户体验设计的方法和流程，包括用户研究、需求分析、原型设计、用户测试等。能够从用户的角度思考问题，设计符合用户需求的产品。

原型设计工具

掌握了原型设计工具，如 Figma、Sketch 等，能够快速创建交互原型，验证设计方案。

移动应用交互

学习了移动应用的交互设计原则，包括触摸界面设计、手势操作、响应式布局等。了解了不同平台（iOS、Android）的设计规范和最佳实践。

性能优化

研究了移动应用的性能优化，能够设计流畅的交互体验，即使在资源有限的设备上也能保持良好的性能。

机器学习基础

学习了机器学习的基本概念和算法，包括监督学习、无监督学习、强化学习等。掌握了常见的机器学习模型，如线性回归、决策树、神经网络等。

数据处理

实践了数据预处理、特征工程、模型训练和评估等完整的机器学习流程，能够解决实际问题。

深度学习

研究了深度学习的核心概念和技术，包括卷积神经网络、循环神经网络、生成对抗网络等。掌握了 TensorFlow、PyTorch 等深度学习框架的使用。

应用领域

探索了深度学习在计算机视觉、自然语言处理等领域的应用，了解了最新的研究进展。

AI在游戏中的应用

学习了 AI 在游戏开发中的应用，包括 NPC 行为设计、路径规划、决策系统等。研究了如何使用机器学习技术创建更智能、更具挑战性的游戏 AI。

强化学习

实践了使用强化学习训练游戏 AI，能够让 AI 通过自我学习不断提高游戏水平。
